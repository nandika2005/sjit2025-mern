Git and Github Commands

  • Git Commands
    1. git init-Initializing the git initially at local repository
    2. git add-Used to add a file to the local repository
    3. git commit -m""-Used to commit the changes made to a file
    4. git push-used to push the modified file in the repository
  • GitHub Commands
    1. git remote add origin ""-Add file from local to a global repo
    2. git commitcommit by comment message
    3. git push origin branchPush to global repository
  • Other Commands
    1. git configto see the entire config list so far
    2. git statusShows the status of the repository
    3. git logshows the entry and manipulated files by whom and when
